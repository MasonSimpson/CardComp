# CardComp

Free sports card comp tool powered by eBay sold listings. No subscription required.

## Stack
- **Frontend:** Vanilla HTML/CSS/JS
- **Backend:** Vercel serverless functions
- **Data:** eBay Browse API (sold listings)

## Project Structure
```
cardcomp/
├── api/
│   └── search.js       ← Vercel serverless function
├── public/
│   └── index.html      ← Frontend
├── vercel.json         ← Routing config
└── README.md
```
